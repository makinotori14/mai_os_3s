\section{Условие}

\textbf{Цель работы:}

\begin{itemize}
    \item изучение механизмов взаимодействия программ с ядром ОС;
    \item анализ работы процессов, потоков и межпроцессного взаимодействия;
    \item освоение практики анализа программ с использованием \texttt{strace};
    \item подтверждение корректности выполнения лабораторных работ
          через системные вызовы.
\end{itemize}

\textbf{Задание:}

Для лабораторных работ №1–4 по курсу \enquote{Операционные системы}
необходимо проанализировать выполнение программ на уровне системных вызовов.

Для каждой лабораторной работы требуется:
\begin{itemize}
    \item определить ключевые системные вызовы;
    \item пояснить назначение каждого вызова;
    \item показать соответствие логики программы требованиям задания;
    \item подтвердить корректность выполнения через трассировку \texttt{strace}.
\end{itemize}
