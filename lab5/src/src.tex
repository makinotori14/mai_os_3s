\section{Метод решения}

Для анализа работы программ использовалась утилита \texttt{strace},
позволяющая отследить системные вызовы, выполняемые процессом
во время его выполнения.

Каждая программа запускалась с тестовыми входными данными,
после чего полученные логи анализировались построчно.

Общий подход:
\begin{itemize}
    \item запуск программы под управлением \texttt{strace};
    \item фиксация системных вызовов;
    \item сопоставление вызовов с логическими этапами работы программы;
    \item анализ корректности управления ресурсами и синхронизации.
\end{itemize}

Анализ проводился отдельно для каждой лабораторной работы.
