\section{Результаты}

В ходе выполнения лабораторной работы была реализована система из двух динамических
библиотек, содержащих различные реализации одинаковых функций.

Были написаны программы:

\begin{itemize}
    \item \textbf{program1} — использует первую библиотеку через статическую линковку;
    \item \textbf{program2} — загружает обе библиотеки динамически и вызывает
          конкретную реализацию в зависимости от типа операции.
\end{itemize}

Программа \textbf{program2} корректно загружает файлы \texttt{.so}, получает
адреса необходимых функций, вызывает их и завершает работу, освобождая все ресурсы.

Функции работают корректно как для вычисления числа $e$, так и для вычисления
площади прямоугольного треугольника.

Фрагменты трассировки системных вызовов (\texttt{strace/dtruss}) будут добавлены отдельно.
